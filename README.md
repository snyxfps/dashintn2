# Central de Integrações

Aplicação web moderna e responsiva para gestão operacional dos serviços:
- Integrações SMP
- Multicadastro
- RC-V
- Tecnologia Logística
- Tecnologia Risco

## Stack
- Vite + React + TypeScript
- Tailwind + shadcn/ui
- Supabase (Auth + Postgres)

## Rodar localmente

Requisitos: Node.js 18+

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
